# FRIDA — First Response Integrated Damage Assessment
## UNDP / InnoCentive Challenge — Build the Future of Crisis Mapping

FRIDA is a Progressive Web App (PWA) designed for crisis damage reporting in conditions of extreme stress, zero connectivity, and physical danger.

## Live Demo
[Open FRIDA →](https://[your-username].github.io/frida-app)

## Files
- `index.html` — Main application (citizen interface, live map, UNDP dashboard)
- `manifest.json` — PWA manifest for installability
- `service-worker.js` — Offline functionality and background sync

## How to deploy to GitHub Pages
1. Create a repository named `frida-app`
2. Upload these three files
3. Go to Settings → Pages → Source: main branch
4. Your app will be live at `https://[username].github.io/frida-app`

## Features demonstrated in this prototype
- Emergency button with passive activation design
- 6-step progressive form with survival recommendations
- Offline storage with sync on reconnection
- Live map with geolocated damage reports
- UNDP dashboard with AI priority ranking
- CSV export in standard formats
- 6 UN language support (EN, ES, FR, AR, ZH, RU)
- Multi-event management
- Credibility-based reputation system
